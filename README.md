# Guia de uso
## En windows
### Crear entorno venv
python -m venv venv

### Entrar en el entorno virtual
venv\Scripts\activate

### Istalar paquetes dentro del entorno virtual
pip install Flask matplotlib plotly numpy

### Ejecutar
python3 app.py

### En un navegador utilizar la ruta:

http://127.0.0.1:5000/

Que es local host de la maquina.
